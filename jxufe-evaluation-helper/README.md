# 江财教学评价填写助手

一个仅适配 `https://jwxt.jxufe.edu.cn/` 的 Manifest V3 Chrome 扩展。它可以扫描未评价课程、批量填写单选题和文字评价、按需逐门暂存，并在最终提交前展示核对清单。已有分数或显示“删除暂存”的课程会被自动跳过。

## 安装

1. 在 Chrome 打开 `chrome://extensions/`。
2. 开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录 `jxufe-evaluation-helper`。

## 使用

1. 登录教务系统并进入“教学评价 > 提交教学评价表”。
2. 点击扩展图标，先使用“扫描课程”确认识别结果。
3. 编辑评分和文字模板。
4. 建议首次关闭“自动暂存”，只测试一门课程；核对无误后再开启自动暂存进行批量处理。
5. 批量处理完成后阅读预览。扩展只会定位并高亮最终“提交”按钮，必须由用户亲手点击。

## 安全边界

- 不读取 Cookie、密码或其他网站内容。
- 不上传课程、教师或学生信息。
- 不处理验证码、登录过期或系统错误。
- 开启自动暂存时，仅自动确认“保存数据但不会完成评价，是否继续”这一类提示。
- 删除、最终提交和其他确认提示不会被自动确认。
- 不自动点击学校系统的最终“提交”按钮。

## 测试

运行纯逻辑测试：

```powershell
node --test tests/shared.test.js
```

`mock/evaluation.html` 提供 10 门课程、16 道单选题和 2 个必填文本框的自动测试夹具。页面完成后会在 `<body>` 上写入 `data-test-status="passed"`；测试同时断言脚本没有点击最终提交。由于扩展的域名权限严格限制在学校站点，模拟页不会被正式扩展自动注入。
